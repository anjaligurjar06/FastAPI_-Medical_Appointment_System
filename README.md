# 🏥 CarePoint Clinic - Medical Appointment System API

A complete **FastAPI backend project** for managing doctors and medical appointments. It supports advanced features like filtering, searching, sorting, pagination, appointment booking, and status management.

---

## 🚀 Features

### 👨‍⚕️ Doctor Management

* View all doctors
* Search, filter, and sort doctors
* Pagination support
* Browse (combined search + sort + pagination)
* Get doctor by ID
* Add, update, delete doctors
* Summary insights (most experienced, cheapest, specialization count)

### 📅 Appointment Management

* Book appointment with dynamic fee calculation
* Confirm, cancel, complete appointments
* View all & active appointments
* Search appointments by patient
* Sort and paginate appointments
* Filter appointments by doctor

### 💡 Fee Calculation Logic

* Video consultation → 20% discount
* Emergency → 50% extra charge
* Senior citizens → additional 15% discount
